# iNotes - FastAPI-MongoDB-Notes-App
iNotes is a simple yet powerful web application designed to help users capture thoughts and notes on the go. Built with FastAPI for high-performance backend logic and MongoDB Atlas for scalable cloud storage.

## Features

* Full CRUD Functionality: Create and Read notes seamlessly.

* FastAPI Backend: Utilizes asynchronous request handling for speed and efficiency.

* MongoDB Integration: Leverages PyMongo to store notes as flexible JSON-like documents.

* Jinja2 Templating: Dynamic rendering of notes on the front-end.

* Responsive UI: Styled with Bootstrap 5 for a clean, mobile-friendly experience.

## Tech Stack

* Framework: FastAPI

* Database: MongoDB (Atlas Cloud)

* Template Engine: Jinja2

* Client Library: PyMongo

* Styling: Bootstrap 5
